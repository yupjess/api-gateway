# API Gateway

Routing layer for PH Transport API and Adventist Directory Scraper API.

## Routes

| Source | Destination |
|--------|-------------|
| `/api/transport/*` | `https://ph-transport-api.vercel.app/api/*`
| `/api/directory/*` | `https://adventist-scraper-api.vercel.app/api/*`

## Deployment

This project is deployed to Vercel.
